\section{WIP: Basics}

\subsubsection{Approval Elections}

Election is a pair \(E = (C, V)\), where 
\(C = \{c_1,...,c_m\}\) is a set of candidates and 
\(V = \{v_1,...,v_n\}\) is a set of voters. 
Each voter \(v \in V\) approves a subset of the candidates and disapproves the remaining candidates.
I will be referring to it as a \mbox{\textit{approval set}} and for each voter $v \in V$ denote it as $A(v)$,
when using $A$ just by itself I refer to all approval sets in the given election (preferences).

\subsubsection{Committee Selection Rules}

Voting rules are algorithms used to select winners (committee) of a given election.
Each rule has different properties and must be selected accordingly to the goal of the election.
For the detailed overview of multiwinner voting rules I point to \cite{faliszewskiSkowronMultiWinnerOverview}.

\section{Preference Restrictions}\label{pref-restrictions-section}

Many social choice problems such as winner determination, or control problem are known to be computationally hard.
To deal with it we can use approximate or parametrized algorithms (WIP: Podać przykłady metod / cytacje).
The alternative approach is to introduce restrictions on the preference domains. (WIP: Cytacje research'u)
In the last few years this area of research advanced a lot, providing multiple efficient algorithms
for the popular committee selection rules, and other problems.

\subsection{Euclidean Preference Restrictions}

\subsubsection{Voter Candidate Range (VCR)}\label{VCR-definition}
Given an election \(E = (C, V)\), we say that the voters have \textit{1D-Euclidean} preferences if
for every agent (voter or candidate) there exists a point $x_a$ and nonnegative real value radius $r_a$ such that:
$\forall \text{ } c \in C \text{ } \forall \text{ } v \in V: c \in A(v) \Leftrightarrow \sqrt{(x_c - x_v)^2} \leq r_c + r_v$

We can easily think of it as: the point $x_a$ represents ideal position 
in the 1-dimensional space of opinions. The radius can be seen as candidate's charisma or
voter's willingness to compromise.

\subsubsection{Candidate Range (CR)}
We say that P satisfies CR if candidates in $C$ can be
ordered so that each of the sets $v_i$ forms an interval of that ordering.
CR is special case of VCR, where we require all the voters to have radii equal to zero

\subsubsection{Voter Range (VR)}
We say that P satisfies VR if the voters in $V$ can be reordered
so that for every candidate $c_i$ the voters that approve c form an interval of that
ordering.
VR is a special case of VCR, where we require all the candidates to have radii equal to zero
\newline 

\subsubsection{Voter Interval (VI) and Candidate Interval (CI)}
\cite{falisz} showed that CR and VR are equivalent to respectively VI and CI
introduced by \cite{DBLP:journals/corr/ElkindL15}.

The VCR model has larger domain than the VR and CR combined.
For example, election with candidate set
$C = \{a, b, c, d\}$ and voters with approval sets $\{a, b\}$, $\{b, c\}$,
$\{b, d\}$, and $\{a, b, c, d\}$ is VCR, but neither VR nor CR.

\subsection{Integer Linear Program}
Integer linear programming (ILP) is a common approach to solve NP-hard problems, 
furthermore it can be used effectively for other problems as well.
(WIP: czy zdanie od furthermore ... jest potrzebne) 
To define a computational problem as an integer linear program,
we have to create a system of linear inequalities (constraints) 
(WIP: Sprawdzic czy na pewno inequalities == constraints) and 
optionally a linear function to optimize, alternatively we can just check feasibility of the constraints.
Next to obtain a solution we have to use an optimization solver.
Popular choices are Gurobi or CPLEX which are state-of-the-art general purpose solvers.
