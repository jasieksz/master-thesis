\begin{abstract}

    We study approval-based elections, where voters' preferences are in the 1D-Euclidean domain.
    Our focus is on preference restrictions, in particular on a recently introduced model: the voter/candidate range,
    defined by \cite{faliszewski-vcr}. 
    We explore multiple areas: First, we create domain detection algorithm based on a reduction to integer linear programming,
    and compare it with existing solutions. Next, we use the said algorithm in an experiment,
    in order to establish the probability with which a random election falls within a certain domain.
    At last, we provide a polynomial-time algorithm for the approval-voting constructive control problem,
    for elections in the one-dimensional voter/candidate range domain.


\end{abstract}
