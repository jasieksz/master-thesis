\section{Profile Structure Detection}
Detection algorithms are used to check whether election profile satisfies desirable structural properties.
For example to check CR (VR) properties it's enough to check for the \textit{consecutive ones property}
in the preferences matrix. (WIP : Citation)

Our focus lies on the one dimensional VCR property. 
Having preferences matrix as input we have to provide a mapping of each agent (voter/candidate)
to the point on the real line and a nonnegative real value representing radius,
so that it satisfies the VCR property.  

\section{Integer Linear Program}
Integer linear programming (ILP) is a common approach to solve NP-hard problems, 
furthermore it can be used effectively for other problems as well.

To define a computational problem as an integer linear program,
we have to create a system of linear inequalities (constraints)

\subsubsection{Transform the VCR definition into linear constraints}
First we have to redefine the VCR definition in order to make it appropriate 
for the ILP. It is based on the euclidean distance which cannot be directly represented in the ILP.
Figure \ref{vcr-to-linear-eq} (WIP: Figure/Equation/sth else?) shows transformation of the VCR definition into 
linear inequalities. (WIP: Czy powtarzam VCR definition za często?)

\begin{equation}\label{vcr-to-linear-eq}
    \begin{split}
    VCR & \iff \sqrt{(x_c - x_v)^2} \le r_c + r_v \\
     & \iff \|x_c - x_v\| \le r_c + r_v \\
     & \iff -(r_c + r_v) \le x_c - x_v \le r_c + r_v \\
     & \iff x_c - x_v \le r_c + r_v\ \ \lor \ \ -(x_c - x_v) \le r_c + r_v
    \end{split}
\end{equation}
The constraint obtained from the equation \ref{vcr-to-linear-eq} 
is not suitable yet for the ILP because it contains logical \textit{OR} condition
which again cannot be directly represented in the ILP. 
However, this issue is quite common and most solvers provide additional built-in mechanisms to solve it.

In Gurobi solver it's called \textit{INDICATOR} constraint
which allows creating versatile constraints in form of:
$z = f \rightarrow a^Tx \leq b$ which states that if the binary indicator variable $z$
is equal to $f$, where $f \in \{0,1\}$, then the linear constraint 
$a^Tx \leq b$ should hold. On the other hand, if $z = 1-f$, 
the linear constraint may be violated.

\subsubsection{Detection algorithm} \label{vcr-detection-ilp-section}
Final formulation of the ILP for recognising VCR property in the election profile 
is shown in the figure \ref{vcr-detection-ilp}.
Variables used by the program $x_a$ and $r_a$ represent agent's point on the real line
and their radius respectively.
Additionally, we use two binary variables $p$ and $q$ required by the indicator constraints.

\end{aligned}
\end{equation}
