\textbf{Approval Elections}
\indent
Election is a pair \(E = (C, V)\) where 
\(C = \{c_1,...,c_m\}\) is a set of candidates and 
\(V = \{v_1,...,v_n\}\) is a list of voters. 
Each voter \(v \in V\) approves a subset of the candidates and disapproves the remaining candidates.

\subsubsection{Preference Restrictions}
One-Dimensional Euclidean Preferences
\cite{DBLP:journals/corr/ElkindL15}

\textbf{Voter Candidate Range}
\indent
Given an election \(E = (C, V)\), we say that the voters have \textit{1D-Euclidean} preferences 
For each agent there exists a point $x_a$ and nonnegative realvalue $r_a$ such that:
$c \in A(v) \Leftrightarrow \sqrt{(x_c - x_v)^2} \leq r_c + r_v$

Intuitevely, for $a \in C \cup V$ the point $x_a$ describes a's ideal position 
in 1-dimmensional space of opinions.
For candidate $c \in C, r_c$ can be seen as c's charisma.
For candidate $v \in V, r_v$ can be seen as v's willingness to compromise.

\textbf{Candidate Range}
\indent
We say that P satisfies CR if candidates in C can be
ordered so that each of the sets vi forms an interval of that ordering.

CR is special case of VCR, where we require all the voters to have radii equal to zero

\textbf{Voter Range}
\indent
We say that P satisfies VR if the voters in P can be reordered
so that for every candidate c the voters that approve c form an interval of that
ordering.

VR is special case of VCR, where we require all the candidates to have radii equal to zero
\newline

The VCR model is strictly more powerful than the VR
and CR ones. For example, election with candidate set
$C = \{a, b, c, d\}$ and voters with approval sets $\{a, b\}$, $\{b, c\}$,
$\{b, d\}$, and $\{a, b, c, d\}$ is VCR, but neither VR nor CR.

\textbf{Minimax Approval Voting (MAV)}
\indent
\cite{minimax-Brams-Kilgour-Sanver}

As close as possible to the preferences of all parties.
By “close” we mean that the maximum distance of the compromise from the position of any state 
is minimal, which we call a minimax outcome.
\newline
\indent
Selects a committee \(W\) that minimizes the maximum Hamming distance
between \(W\) and voters’ approval ballots. We define MAV-score of a committee \(W\) 
as \(Max(f(W,v1),f(W,v2),...,f(W,vn))\) where \(f(W,vi)=\).
MAV returns a committee \(W\) with the lowest MAV-score.
