\section{Hamming Distance Minimax Approval Voting}

\subsubsection{Hamming Distance}
Given two sets $X,Y$ their Hamming distance is 
$d_{Ham}(X,Y) = |X \setminus Y| + |Y \setminus X|$.
Alternatively for vector representation we define Hamming distance between
vectors of the same size $a,b$ as $d_{Ham}(a,b) = |a \oplus b|$.

\subsubsection{Minimax Approval Voting (MAV)}
Scoring rule introduced by \cite{minimax-Brams-Kilgour-Sanver}.
Selects a committees $W$ that minimizes the maximum Hamming distance
between \(W\) and voters' approval ballots, i.e. 
$min \text{ } max_{v \in V} \text{ } d_{Ham}(W,A(v))$

\subsubsection{Brute force algorithm}

\begin{verbatim}
def hammingDistance(a: np.array, b: np.array) -> int:
return (a != b).nonzero()[0].shape[0]

def mavScore(profile: Profile, committee:np.array) -> int:
    partialHD = lambda preference: hammingDistance(committee, preference)
    return max([partialHD(pref) for pref in profile.A])

def minimax(profile: Profile, k: int) -> np.array:
    candidateCount = len(profile.C)
    scores = [(mavScore(profile, np.array(committee)), profile) 
                for committee in combinations(range(candidateCount), k)]
    scores.sort(key=lambda mav_p: mav_p[0])
    return scores
\end{verbatim}
