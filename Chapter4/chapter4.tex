\section{Integer Linear Programming}
The only input is \textit{approval set} of each voter. Represented by $2D$ matrix $A_{vc}$

\subsubsection{Variables}

$x_c$ - candidate $c$ position on $x$ axis \\
$x_v$ - voter $v$ position on $x$ axis \\
$r_c$ - candidate $c$ radius \\
$r_v$ - voter $v$ radius 

\subsubsection{Transform VCR condition to linear constraints}

VCR definition formula contains square root and power elements (WIP: Inne słowo).
Hence it's not directly suitable for ILP formulation. Let's transform it to linear constraint.

\begin{equation}
    \begin{split}
    VCR & = \\
     & = \sqrt{(x_c - x_v)^2} \le r_c + r_v \\
     & = \|x_c - x_v\| \le r_c + r_v \\
     & = -(r_c + r_v) \le x_c - x_v \le r_c + r_v \\
     & \iff x_c - x_v \le r_c + r_v\ \ \lor \ \ -(x_c - x_v) \le r_c + r_v
    \end{split}
\end{equation}

When used in the constraints themselves, one of the many uses of Big M,
for example, refers to ensuring equality of variables only when a certain binary variable takes on one value,
but to leave the variables "open" if the binary variable takes on its opposite value.
One instance of this is as follows: for a sufficiently large $M$ and $z$ binary variable (0 or 1), the constraints

\begin{align}
    x - y \leq Mz \\
    x - y \geq -Mz
\end{align}

ensure that $z = 0 \rightarrow x = y$, otherwise $z = 1 \rightarrow -M \leq x - y \leq M$
indicating that the variables x and y can have any
values so long as the absolute value of their difference 
is bounded by M (hence the need for M to be "large enough.") 

\subsubsection{ILP detection algorithm}

\noindent Introduce large constant positive parameter $M$\\
Introduce new binary variables for each pair of candidate and voter:\\
$z1_{cv}\\ z2_{cv}$

\begin{align*}
\textit{Minimize 0} \\ 
\textit{foreach (c,v) in } C \times V : \\\textit{if } c \in ApprovalSet(v) \textit{ add constraints} \\ 
x_c - x_v \le r_c + r_v \\
-(x_c - x_v) \le r_c + r_v \\
\textit{if } c \not\in ApprovalSet(v) \textit{ add constraints} \\
x_c - x_v  + M(1 - z1_{cv}) \ge r_c + r_v + 1 \\
-(x_c - x_v) + M(1 - z2_{cv}) \ge r_c + r_v + 1 \\
z1_{cv} + z2_{cv} >= 1
\end{align*}
